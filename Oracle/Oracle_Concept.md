> # Oracle Setting File

>> ## Listener

- 파일명 : `listener.ora`
- 경로 확인 : `# lsnrctl status`
- Default Path : `$ORACLE_HOME/network/admin/listener.ora`
- 파일 기능 : client가 Oracle server에 접속하기 위해서 server 컴퓨터에 하는 설정 파일
